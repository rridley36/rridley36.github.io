Overview

This project demonstrates a real-time application developed for the EEL 5862 Real-Time Systems course. It combines a FreeRTOS-based task scheduler with peripheral I/O on an ESP32-S3 development board, coordinating multiple concurrent tasks under strict timing constraints.

Platform ESP32-S3 DevKitM-1
Framework Arduino + FreeRTOS
Toolchain PlatformIO
Language C / C++

Demo Video

A short walkthrough of the working project.

Features

  • Concurrent tasks scheduled via FreeRTOS primitives.
  • Inter-task communication using queues and semaphores.
  • Deterministic timing verified against hard deadlines.
  • Peripheral integration over GPIO, I²C, and SPI.
