Overview

Developed a healthcare-focused real-time patient monitoring system on an ESP32 using FreeRTOS, simulating heart rate data through an ADC input to model vital sign acquisition. The system detects abnormal heart rate conditions and triggers clinical-style alerts using LEDs, a PWM-driven buzzer, and a nurse call interface for immediate response. Concurrency is handled with semaphores and mutexes to ensure reliable coordination between sensing, alerting, and user interaction tasks. This project reflects embedded system design principles used in medical devices, emphasizing deterministic timing, fault awareness, and responsive patient care support.

Platform ESP32

Features

  • Concurrent tasks scheduled via FreeRTOS for sensing, monitoring, and alert handling.
  • Inter-task communication using semaphores and mutexes for safe event signaling and shared resource access.
  • Deterministic timing maintained using vTaskDelayUntil() for consistent real-time behavior.
  • Healthcare alert system with threshold-based heart rate detection triggering LEDs and a buzzer alarm.
  • Sensor integration using ADC input to simulate real-time heart rate data.
  • User interaction through a debounced nurse call button for manual assistance requests.
  • Reliable output control with mutex-protected serial logging and coordinated alert responses.
